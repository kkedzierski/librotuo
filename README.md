# Librotuo - book searcher
### website for finding books and check availability for downloading or buy

### Technologies

***

* gatsby: 2.32.3
* react: 16.14.0
* HTML
* CSS

### Instalation

***

After pull respository to open project on localhost write in console:  
```gatsby develop```
